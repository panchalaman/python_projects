# Python Projects

A collection of completed Python projects built throughout my 100-days coding challenge.

## Projects

- **Blackjack Game** — Full terminal-based card game with dealer AI
- **Password Generator** — Secure random password generation tool
- **Quiz App** — Interactive quiz game using Open Trivia API
- **Snake Game** — Classic snake game built with Turtle graphics
- **Pomodoro Timer** — Productivity timer with GUI
- **NATO Alphabet Converter** — Phonetic alphabet converter

## Tech Stack

- Python 3
- Tkinter (GUI)
- Turtle Graphics
- APIs (requests library)

## Getting Started

```bash
git clone https://github.com/panchalaman/python_projects.git
cd python_projects
pip install -r requirements.txt
python <project_name>.py
```
